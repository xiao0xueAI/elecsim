# ElecSim Studio Pro

**电工电路仿真与工作原理可视化** | v2026.05.06

一个基于浏览器的交互式电路仿真工具，支持拖放搭建电路、实时仿真计算、电流可视化动画和工作原理导出。

## ✨ 功能特性

- **元件库**：电池、交流电源、电阻、电容、电感、开关、LED、二极管、NPN三极管、蜂鸣器、直流电机、电压表、电流表
- **电路画布**：网格对齐、滚轮缩放、Alt+拖动平移
- **连线系统**：点击端子连线，L形自动走线
- **实时仿真**：基于基尔霍夫定律的简化电路分析，实时显示电压/电流/功率
- **可视化效果**：LED发光、电机旋转、蜂鸣器振动、电流方向箭头动画
- **X光视图**：透视查看元件内部结构
- **导出动画**：支持 720p/1080p/4K 分辨率，5-30秒时长选择
- **自定义元件**：通过表单添加自定义元件，JSON定义电气属性

## 🚀 使用方法

直接在浏览器中打开 `index.html` 即可使用，无需安装任何依赖。

## 🎮 操作指南

| 操作 | 说明 |
|------|------|
| 拖放元件 | 从左侧元件库拖放到画布，或双击添加 |
| 连线 | 点击一个元件端子，再点击另一个端子完成连线 |
| 运行仿真 | 点击顶部「运行仿真」按钮 |
| 修改参数 | 点击元件，在右侧属性面板修改 |
| 删除元件 | 右键点击元件，或选中后按 Delete 键 |
| 取消连线 | 按 Escape 键 |
| 缩放 | 鼠标滚轮 |
| 平移 | Alt + 鼠标拖动 |

## 🛠️ 技术栈

- 纯前端 HTML5 + Canvas + JavaScript
- 零依赖，单文件应用
- 响应式设计，适配多种屏幕
