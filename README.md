# ElecSim Studio Pro v2.0

**QIACHIP 电工电路仿真平台** | 专业级电路仿真与产品演示工具

## 🔗 在线预览

**ElecSim Studio Pro v2.0** | https://xiao0xueai.github.io/elecsim/

基于浏览器的交互式电路仿真工具，专注遥控模块/控制器场景。支持拖放搭建电路、实时仿真、继电器行为模拟、故障检测、QIACHIP 产品集成演示。

## 功能特性

### 元件系统
- **30+ 内置元件**：电源(3)、开关/按钮(5)、保护器件(2)、继电器/接触器(3)、传感器(5)、输出器件(6)、仪表(2)、无源元件(5)
- **多引脚模型**：5针/8针继电器、交流接触器等
- **自定义元件**：表单添加自定义元件

### 仿真引擎
- **Union-Find 图分析**：支持串联 + 并联电路
- **继电器行为**：线圈通电→COM切换到NO，断电→回到NC
- **故障检测**：短路/断路/过流实时诊断
- **电流粒子可视化**：速度和颜色反映电流状态

### QIACHIP 产品集成
- **内置产品模板**：1路/2路/4路无线继电器模块、WiFi继电器
- **产品上传**：图片 + 可视化引脚编辑器 + 电气参数
- **产品目录**：左侧产品面板，拖放使用
- **电路模板**：预设 QIACHIP 应用电路

### UI 功能
- **拐点布线**：点击添加路径点，灵活走线
- **撤销/重做**：Ctrl+Z / Ctrl+Y
- **保存/加载**：自动保存 + JSON 导入导出
- **右键菜单**：删除、复制、属性
- **X光视图**：透视查看内部结构
- **视频导出**：720p/1080p/4K WebM

## 使用方法

直接在浏览器中打开 `index.html` 即可使用，无需安装任何依赖。

## 操作指南

| 操作 | 说明 |
|------|------|
| 拖放元件 | 从左侧元件库拖放到画布，或双击添加 |
| 连线 | 点击引脚开始，点击目标引脚完成；点击空白添加拐点 |
| 运行仿真 | 点击顶部「运行仿真」按钮 |
| 修改参数 | 点击元件，在右侧属性面板修改 |
| 删除元件 | 右键→删除，或选中后按 Delete |
| 取消连线 | 按 Escape |
| 缩放 | 鼠标滚轮 |
| 平移 | Alt + 拖动 或 鼠标中键拖动 |
| 撤销 | Ctrl+Z |
| 重做 | Ctrl+Y |
| 保存 | Ctrl+S |
| 适应全部 | 双击空白处 |
| 切换标签 | 左侧 元件库/产品/模板 |

## 技术栈

- 纯前端 HTML5 + Canvas + JavaScript
- 零依赖，单文件应用 (~1800行)
- IIFE 命名空间组织
- 离线可用，加载后无需网络
